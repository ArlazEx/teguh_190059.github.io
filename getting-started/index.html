    Turunan Numerik

    Motode turunan numerik sangat membantu untuk mencari persamaan turunannya. Sebab bila suatu persamaan hanya seperti

    \[ f\left(x\right)=3x^{2}+2x \]

    maka akan sangat mudah untuk menemukan persamaan turunannya. Akan tetapi bagaimana bila suatu persaam seperti berikut

    \[ f\left( x\right) =3e^{2-1x}+4x \]

    maka akan kesusahan bila mencari persamaan turunannya. Maka dari itu digunakanlah motode numerik untuk mencari persamaan turunan tersebut.

    Motode yang dapat digunakan meliputi beda maju, beda mundur dan beda pusat.

    Turunan ke-1¶

    Untuk turunan ke 1 kita ibaratkan x0 sebagai titik awal, maka untuk rumus turunan dari 3 metode tersebut adalah

    Metode Beda Maju¶

    Metode beda maju berbentuk seperti grafik berikut

    Untuk rumusnya : $$ f'\left( x_{0}\right) =\dfrac{f\left( x_{1}\right) -f\left( x_{0}\right) }{\left( x_{1}-x_{0}\right)} $$

    Metode Beda Mundur¶

    Metode beda mundur berbentuk seperti grafik berikut

    Untuk rumusnya : $$ f'\left( x_{0}\right) =\dfrac{f\left( x_{0}\right) -f\left( x_{-1}\right) }{\left( x_{0}-x_{-1}\right) } $$

    Metode Beda Pusat¶

    Metode beda pusat berbentuk seperti grafik berikut

    Untuk rumusnya : $$ f'\left( x_{0}\right) =\dfrac{f\left( x_{1}\right) -f\left( x_{-1}\right) }{\left( x_{1}-x_{-1}\right) } $$

    Turunan ke-2¶

    Turunan ke-2 dapat diperoleh dengan metode taylor. Turunan ke-2 sendiri merupakan turunan dari turunan suatu fungsi.

    Berikut merupakan rumus turunan ke-2.

    Metode Beda Maju¶

    \[ f''\left( x_{0}\right) =\dfrac{\left( f\left( x_{2}\right) -2f\left( x_{1}\right) +f\left( x_{0}\right) \right) }{\left( x_{1}-x_{0}\right) ^{2}} \]

    Metode Beda Mundur¶

    \[ f''\left( x_{0}\right) =\dfrac{\left( f\left( x_{-2}\right) -2f\left( x_{-1}\right) +f\left( x_{0}\right) \right) }{\left( x_{-1}-x_{0}\right) ^{2}} \]

    Metode Beda Pusat¶

    \[ f''\left( x_{0}\right) =\dfrac{\left( f\left( x_{1}\right) -2f\left( x_{0}\right) +f\left( x_{-1}\right) \right) }{\left( x_{0}-x_{-1}\right) ^{2}} \]

    Jawaban¶

    Untuk xi di 0.1 karena tidak ada nilai sebelum 0.1 maka hanya dapat menggunakan metode maju saja Sedangakan untuk xi di 0.3 bisa menggunakan semua metode.

    Turunan ke-1 untuk xi dititik 0.1:

    -Beda maju: $$ \begin{aligned}f'\left( 0.1\right) =\dfrac{( f\left( 0.2\right) -f\left( 0.1\right) }{\left( 0.2-0.1\right) }\ =\dfrac{\left( 0.92-043\right) }{0.1}=4.9\end{aligned} $$

    Turunan ke-1 untuk xi dititik 0.3:

    -Beda maju: $$ \begin{aligned}f'\left( 0.3\right) =\dfrac{( f\left( 0.4\right) -f\left( 0.3\right) }{\left( 0.4-0.3\right) }\ =\dfrac{\left( 2.08-1.47\right) }{0.1}=6.1\end{aligned} $$

    -Beda mundur: $$ \begin{aligned}f'\left( 0.3\right) =\dfrac{( f\left( 0.3\right) -f\left( 0.2\right) }{\left( 0.3-0.2\right) }\ =\dfrac{\left( 1.47-0.92\right) }{0.1}=5.5\end{aligned} $$

    -Beda pusat: $$ \begin{aligned}f'\left( 0.3\right) =\dfrac{( f\left( 0.4\right) -f\left( 0.2\right) }{\left( 0.4-0.2\right) }\ =\dfrac{\left( 2.08-0.92\right) }{0.1}=5.8\end{aligned} $$

    Jika ingin mencari turunan ke dua maka prosesnya hampir sama, akan tetapi rumusnya berbeda.

    Code Program¶

    Code:

    tabel_soal = [[0.1,0.43], [0.2,0.92], [0.3,1.47],[0.4,2.08],[0.5,2.75]]

def hitung(a):
    print("Turunan pertama dari xi =",a)
    if a == tabel_soal[0][0]:
        x = (tabel_soal[1][1]-tabel_soal[0][1])/(tabel_soal[1][0]-tabel_soal[0][0])
        print("Dengan Metode Maju hasilnya = ",x )
    elif a == tabel_soal[len(tabel_soal)-1][0]:
        x = (tabel_soal[len(tabel_soal)-1][1]-tabel_soal[len(tabel_soal)-2][1])/(tabel_soal[len(tabel_soal)-1][0]-tabel_soal[len(tabel_soal)-2][0])
        print("Dengan Metode Mundur hasilnya = ",x)
    else:
        for i in range(len(tabel_soal)):
            if a == tabel_soal[i][0]:
                x = (tabel_soal[i+1][1]-tabel_soal[i][1])/(tabel_soal[i+1][0]-tabel_soal[i][0])
                print("Dengan Metode Maju hasilnya = ",x )
                y = (tabel_soal[i][1]-tabel_soal[i-1][1])/(tabel_soal[i][0]-tabel_soal[i-1][0])
                print("Dengan Metode Mundur hasilnya = ",y)
                z = (tabel_soal[i+1][1]-tabel_soal[i-1][1])/(tabel_soal[i+1][0]-tabel_soal[i-1][0])
                print("Dengan Metode Mundur hasilnya = ",z)
    print("")


hitung(0.3)
hitung(0.1)

print("Sedangkan untuk xi=0.5 :")
hitung(0.5)
